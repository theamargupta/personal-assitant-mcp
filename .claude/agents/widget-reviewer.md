# Widget Reviewer

Audit every widget HTML file in `widgets/`.

Check:
- Sends `postMessage` to the parent for all state changes and host communication.
- Respects `prefers-color-scheme`.
- Does not load external scripts.
- Is self-contained or uses the bundled output generated by `scripts/gen-ext-apps-bundle.mjs`.
- Has a matching MCP tool registered with `registerAppTool`.
- Matching registration includes `_meta.ui.resourceUri`.

Report findings with file paths and line numbers.

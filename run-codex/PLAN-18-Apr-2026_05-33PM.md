# PLAN — Landing Page Narrative Rewrite (Option B)

## Objective
Rewrite the Sathi landing page so a first-time visitor, within 60 seconds of scrolling, understands:
1. **WHAT Sathi is** — an MCP server that plugs their life (habits, tasks, money, docs, goals, memory) into Claude and ChatGPT, with live widgets, across web and mobile.
2. **HOW to connect it** — exact setup steps for Claude.ai, ChatGPT, desktop MCP clients, the web dashboard, and the iOS app (with the real MCP URL `https://sathi.devfrend.com/api/mcp`).
3. **WHAT they can do** — shown through real screenshots of the three surfaces plus a Hinglish prompt gallery.

The current landing page vaguely gestures at all three. This plan makes each explicit.
